# Galvanize Web Development Immersive

Welcome to the Web Development Immersive!

## Your Mission

Your mission here is to **learn how to solve problems well enough to get and keep a job as a web developer**.

What does it take to get and keep a job?  Well, lots of things, but at the very least you'll need to be able to:

- Be a good problem solver
- Be very productive and get things done (GTD)
- Be able to learn new things quickly

For a more detailed breakdown of what this means see the following:

![](https://students-gschool-production.s3.amazonaws.com/uploads/asset/file/379/wdi-curriculum-overview.png)

Some things you'll learn are [applicable to any programming language](./Overview.md).

Some things you'll learn are [specific implementations](./Implementations.md).
